Package List
============

1. original.py
Uses a model with Legendre convolution layer, feature distillation and attention mechanism.

2. chebyshev_model.py
Uses a model with Chebyshev convolution layer, feature distillation and attention mechanism.

3. no_attention_model.py
Uses a model with Legendre convolution layer, feature distillation but NO attention mechanism.

4. no_distillation_model.py
Uses a model with Legendre convolution layer and attention mechanism but NO feature distillation.


Environment
===========
GPU: NVIDIA 4080 16GB
RAM: 64GB
OS: ubuntu 18.04
PyTorch 1.16.10
Python 3.9.0
